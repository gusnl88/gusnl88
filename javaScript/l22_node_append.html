node 생성과 추가하기

html 문서가 화면에 그대로 출력되고 있다는 착각을 많이 하는데 화면에 출력하고 있는 것은 node객체

document.createNode(TagName); : 요소 노드 개체를 생성하는 함수

document.createTextNode(""): 텍스트 노드 개체를 생성하는 함수

노드 개체를 추가해주는 함수

  • Node.append(node):해당 노드 자식 마지막으로
  • Node.prepend(node):해당 노드 자식 처음으로
  • Node.after(node):해당 노드 맨뒤에
  • Node.before(node):해당 노드 맨앞에
