타입이 명확하지 않아서 발생하는 문제 해결을 위한 const 선언

  • 자바의 변수 : ArrayList List=new ArrayList();
  • js의 변수 : let list=new Array(); list =13; (o 변수는 모든 타입을 참조가능)
  • list Array 자료ㅕ형이라 생각하고 있다가... 13으로 바뀐줄 모르고 list[3]를 참조하면 undefined 가 반환됨
  • 개발자들이 이런 상황을 피하고 싶어서 모든 자료형을 선언할 때 상수(const)로 선언하는 버릇이 있다.
  • 강제가 아니라 선택으로 필요에 따라서 상수로 선언하면 된다.

배열의 내부 반복문 (Iteration 함수 ) 숙지하세요!!

콜백함수 선언시 Iterator 객체를 이용해 배열의 모든 필드에서 콜백 함수를 실행하는 반복문

내부 반복문은 기본 함수를 정의하기 때문에 재사용성이 높고 가독성도 좋다.

  • *이 제일 많이 사용!! every,some 도 그나마 사용.
  • *forEach((value,index,array)=>{}); : 각 요소를 검사하거나 소비하는 반복문
  • *map((value,index,array)=>modify value); return new Array() :각 요소의 값을 변경한 새로운 배열을 반환
  • *reduce((total,value,index,array)=>total 연산 value , total=val ); return total: 모든 요소를 연산한 하나의 결과를 반환
  • *filter((value,index,array)=>(value 비교연산); return new Array() : 조건에 맞는 요소만 남긴 새로운 배열을 반환
  • every((value,index,array)=>(value 비교연산); return boolean :
  • some((value,index,array)=>(value 비교연산); return boolean :
  • find((value,index,array)=>(value 비교연산); return 찾은요소 :
  • findIndex((value,index,array)=>(value 비교연산); return 찾은 요소의 index :
  • includes(찾는 값); return boolean :
  • indexOf(찾는 값, start index=0); return index :
  • lastIndexOf(찾는 값,start index=0); return index:

배열의 정렬 함수(쓸일이 그닥...db oder by 에서 이미정렬...)

배열에 있는 요소가 정렬이 될때까지 반복하는 함수 기본은 문자열을 정렬(코테정도에서 쓸듯)

  • Array.sort() : 문자열 오름차순(asc) 정렬
  • Array.reverse() : 문자열 내림차순(desc) 정렬
  • Array.sort((a,b)=>a-b): 수의 오름차순 정렬
  • Array.sort((a,b)=>b-a): 수의 내림차순 정렬
  • Object.sort((a,b)=>a.필드 -b.필드) : 자료형의 필드의 값으로 오름차순(asc) 정렬
  • Object.sort((a,b)=>a.필드 -b.필드) : 자료형의 필드의 값으로 내림차순(desc) 정렬

Array 차입과 배열 자료형들

브라우저에서 js가 실행될때 브라우저가 필요에 의해서 생성하는 배열들이 존재한다. 이 배열의 타입이 Array 가 아니다.

  • 배열 :자료에 순서가 있고(index) 길이가 있는 자료구조(포괄적인 의미)
  • ja.Array :자바스크립트가 배열을 쉽게 선언하고 다룰 수 있도록 만든 자료형
  • java.new 타입[] : 자바에서 index 와 길이만 있는 배열 타입 (보통 Array 라 부른다,길이변경불가능,가볍다,기본형가능)
  • java.util.ArrayList : Array 가 사용하기 불편해서 지원하려고 나온 자료형 (컬렉션 프레임워크,길이변경가능.자료형만 가능,무겁다)
  • 브라우저.HTMLCollection : 요소(node)를 배열로 다루기 위해 최초로 등장한 자료형
  • 브라우저.NodeList : node 검색이 불편해서 jQuery 처럼 css 선택자로 검색가능한 함수 querySelectorAll 을 추가하면서 만든 배열
  • 브라우저.DOMTokenList : classList 가 반환하는 배열로 class 를 다루기 위한 필드를 제공
    • add() : 요소를 추가하는 함수
    • remove() :요소를 제거하는 함수
    • toggle() :요소의 클레스를 추가 또는 제거를 전환하는 함수
    • contains() :해당 클레스가 있는지 확인
    • forEach() :각 요소를 검사하거나 소비하는 반복문
    • value : ==className 의 값을 의미
  • 브라우저.RadioNodeList : input type 이 radio 와 checkbox 인 node 를 검색하면 반환되는 배열로 type 이 radio 인 경우 선택한 value 가 반환된다.
